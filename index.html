Guides

Recommendations

  • Keep Xcode stay up-to-date.
  • Install Xcode command line tools.
  • Install Ruby via Rbevn or RVM.
  • Install HomeBrew for Mac.
  • Try to use command line if you conformable.

Code stegies

Avoid to fork the repo.

  • all issues to be posted only on main repo (this repo)
  • to make a feature branch easy to test by swtich/checkout the feature branch
  • easy to keep feature up-to-date with master.

Avoid to code on branch which posted as pull request if:

  • You not own the branch.
  • The branch not shared to you.
  • Code note relate to.

Write a feature, bug fixes.

Keep master branch up-to-date

Create a local branch based off master

Using Terminal:

// 1. Checkout master
git checkout master
// 2. Keep master up-to-date
git pull
// 3. create a branch and switch to the branch
git checkout -b <branch-name>

Using Github for Mac

  1. Goto Branch tab.
  2. Double-click on master
  3. Sync branch
  4. Click '+' to create branch based off master
