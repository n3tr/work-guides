Guides

A guide for getting things done, maintaining the project, create a feature.

Recommendations

  • Keep Xcode stay up-to-date.
  • Install Xcode command line tools.
  • Install Ruby via Rbevn or RVM.
  • Install HomeBrew for Mac.
  • Try to use command line if you conformable.

Avoid to fork the repo.

  • All issues to be posted only on main repo (this repo)
  • Push a feature branch is easy to testing by checkout.
  • Easy to keep feature branch up-to-date with master.

Git strategies

Keeping a simple process

  • Create branch based off master.
  • Hack on the branch, do a great stuff.
  • Push whenever, let other help.
  • Pull request for review and test.
  • Merge back to master. (let click merge button if available)

Write a feature, bug fix.

Keeping master branch up-to-date and create a local branch based off master.

Using Terminal:

    # 1. Checkout master
    git checkout master
    # 2. Keep master up-to-date
    git pull
    # 3. create a branch and switch to the branch
    git checkout -b <branch-name>

Using Github for Mac

  1. Goto Branch tab.
  2. Double-click on master
  3. Sync branch
  4. Click '+' to create branch based off master
